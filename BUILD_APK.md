# Как собрать APK для Android

## Способ через GitHub Actions (рекомендуется)

### Шаг 1 — Загрузите проект на GitHub
1. Создайте репозиторий на https://github.com/new
2. Загрузите код (или дайте агенту доступ)

### Шаг 2 — Получите EXPO_TOKEN
1. Войдите на https://expo.dev (аккаунт: rabotnik1009)
2. Перейдите: Account Settings → Access Tokens → Create
3. Скопируйте токен

### Шаг 3 — Добавьте секрет в GitHub
1. В репозитории: Settings → Secrets and variables → Actions
2. Нажмите "New repository secret"
3. Имя: `EXPO_TOKEN`, значение: вставьте токен

### Шаг 4 — Запустите сборку
- Сборка запустится **автоматически** при каждом push в main/master
- Или вручную: Actions → Build APK → Run workflow

### Шаг 5 — Скачайте APK
- После завершения (10–20 минут) зайдите в Actions → последний run
- Найдите раздел "Artifacts" → скачайте `school-diary-apk`
- Внутри zip будет файл `school-diary.apk`

## Установка на телефон
1. На телефоне включите: Настройки → Безопасность → "Неизвестные источники"
2. Перенесите APK на телефон (через USB, почту или мессенджер)
3. Откройте файл → Установить
